# node-serialoscd

Installation: `npm install -g hsgw/node-serialoscd`

Usage: `serialoscd MONOME_TTY`

Options:

* `-V`, `--version`: output the version number
* `-d`, `--debug`: show debugging information
* `-h`, `--help`: output usage information
